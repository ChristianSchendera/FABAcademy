Programming

Let us start with a few programming definitions

void setup()

{

pinMode(13, OUTPUT);

}

void loop()

{

digitalWrite(13, HIGH);

delay(1000);

digitalWrite(13, LOW);

delay(1000);

}

here the Setup starts

the sections of the setup starts here.

Pin 13 should be an output.

section where setup ends.

main program starts

programming section starts.

HIGH for giving power to PIN13 (LED on).

wait 1000 milli-seconds (one second).

LOW for cutting power of PIN13 (LED off).

wait 1000 milli-seconds (one second).

programming section ends
