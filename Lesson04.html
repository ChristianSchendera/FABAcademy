Design

  • IC1 - The Brain - is the head of the IPS and consists of several chips
  • XTAL1 - also called Chrystal - ist the clock
  • R1-6 - are Resistors - they determine the resistence flowing through the IPS
  • D1-2 - are Diods - they ensure, that current only flows in one direction
  • C1-4 - are Capacitors - They are transforming AC to DC

    From image to machine code

    The file our machine can read is a .gcode
    I am using the site fabmodules.org to generate the g.code by uploading the file and determining the specific instructions of our machine and how it should be milled

    First insert the png
    Our machine is the Roland mill which uses the file format .rml
    And select PCB traces (1/64)

    I set the following instructions:

    machine output - MDX-40
    speed in mm/s - 4
    zjog - 5mm
    cut depth - 0.0
    tool diameter - 0.2mm
    number of offsets - -1

    Same for the outter layer

    But the cut depth is 0.1mm
    tool diameter 0.4mm
    number of offsets - 4
    offset overlap - 50%

    Milling the conductive plate

    I plugged in the USB device and drop the file on the computer of the machine.
    First I wanted to mill the inlay, so i needed to change my tool into a diameter of 0.2mm. The screw can be opened with 2 keys

    Now I can set my starting point
    I do that with the arrow keys which controlls the tool inside our machine. With apply I can fix this point to make sure I can later start again at the same point. The rpm is set to 15,000.
    To determine the Z-Axis I need to make sure, that the tool diamter slightly touches the conductive plate. I used a multimeter to get an audio signal when both touches. When the instructions are done, insert the file into the programm and press CUT

    Placing the Components

    For my FABISP I am using the following Components

  • ATTiny 44
  • XTAL 1
  • 2* 49.9 Ohm Resistors
  • 499 Ohm Resitor
  • 2* 10k Ohm Resistors
  • 0 Ohm Resistor
  • 2* 3.3 Diods
  • 2* 1u Capacitors
  • 2* 8p Capacitors
